# Research Sources

Use web mechanics as vocabulary and sanity checks, not as proof of this jar's
binary layout.

- Final Fantasy Wiki, `Stat growth`: summarizes FF1 class-dependent stat growth,
  HP growth, guaranteed stat increases, and Dawn of Souls onward differences.
- Final Fantasy Wiki, `Final Fantasy stats`: summarizes FF1 stats, HP/MP caps,
  and common stat meanings.
- GameFAQs, `Final Fantasy - Game Mechanics Guide` by AstralEsper: documents NES
  formulas and class growth tables, useful for comparing port behavior.

Confirmed-local evidence should override all generic FF1 references.
